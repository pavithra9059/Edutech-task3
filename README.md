User Management System (JavaScript ES6+)

The User Management System is a simple and interactive frontend web application built using HTML, CSS, and JavaScript (ES6+).
It allows users to perform basic CRUD operations such as adding, editing, deleting, and searching users dynamically.
The project focuses on DOM manipulation and event handling to create a responsive and real-time user experience.
It is designed as an FSD internship project to strengthen core JavaScript and frontend development skills.

* Technologies Used
->HTML5 for structure
->CSS3 for styling and layout
->JavaScript ES6+ for functionality
->DOM manipulation for dynamic updates
->Array methods like push, find, and filter

* Learning Outcomes
This project helps in understanding how frontend applications work in real time.
It improves skills in JavaScript logic building, handling user input, updating UI dynamically, and managing data without using any frameworks or backend.

->Learn CRUD operations in frontend
->Understand event handling in JavaScript
->Improve DOM manipulation skills
->Practice ES6+ JavaScript features
->Build responsive UI components
